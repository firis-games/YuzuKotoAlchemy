ゆづことアルケミー
===

Minecraft 1.12.2 ProjectEアドオンMODです。

---

## 概要
ニコニコ動画の「結月さんと茜の錬金術師」で作成したProjectEのアドオンです。  

## 注意事項

- Minecraft 1.12.2
- Minecraft Forge 1.12.2-14.23.5.2768
- ProjectE-1.12.2-PE1.4.1
